# 📂 Understanding Excel Workbooks

In Microsoft Excel:

## Workbook

A **workbook** is the complete Excel file that contains one or more worksheets.

Example:

```

Sales_Report.xlsx

```

A workbook may contain:

- Sales data worksheet
- Customer worksheet
- Summary dashboard worksheet
- Analysis worksheet

---

## Worksheet

A worksheet is an individual sheet inside a workbook where data is stored and analyzed.

Example:

Workbook:

```

Company_Report.xlsx

```
├── Sales Data
├── Customer Data
├── Dashboard
```

```

# 📌 Save vs Save As

Understanding the difference between Save and Save As is important.

---

# Save

The Save option updates the existing workbook.

Example:

Original file:

```

Sales_Report.xlsx

```

After editing:

```

Save

```

The same file is updated.

---

# Save As

Save As creates a new copy of the workbook.

Example:

Original:

```

Sales_Report.xlsx

```

Create:

```

Sales_Report_Backup.xlsx

```

Purpose:

- Create backups.
- Maintain different versions.
- Preserve original files.

---

# 🔄 Creating Workbook Copies

The instructor explains that sometimes analysts need duplicate files.

Reasons:

- Backup previous work.
- Test changes safely.
- Maintain project history.

Example:

Original:

```

Monthly_Sales_Report.xlsx

```

Copy:

```

Monthly_Sales_Report_Test.xlsx

```

This allows experimentation without damaging the original file.

# 📄 CSV Format (Comma Separated Values)

CSV is a commonly used format in Data Analytics.

CSV means:

**Comma Separated Values**

Example:

```

Name,Age,Department

Ali,25,Finance

Sara,30,Marketing

```

---

# Importance of CSV in Analytics

CSV files are widely used because:

- They are lightweight.
- They can be imported into different tools.
- They work with databases.
- They are supported by Power BI and SQL tools.

Examples:

- Importing sales data into Power BI.
- Loading datasets into databases.
- Sharing structured information.

---

# 💼 Business Example

A company receives customer data from different systems.

The analyst may:

1. Export Excel data as CSV.
2. Load CSV into Power BI.
3. Clean and transform the data.
4. Create dashboards.

# 🎯 Interview Preparation

## Beginner Questions

### Q1. What is the difference between a workbook and worksheet?

**Answer:**

A workbook is the complete Excel file, while worksheets are individual pages inside that workbook where data is stored.

---

### Q2. What is the difference between Save and Save As?

**Answer:**

Save updates the existing file, while Save As creates a new copy with a different name or location.

---

### Q3. Why is CSV important in Data Analytics?

**Answer:**

CSV is a lightweight data format widely used for transferring data between Excel, databases, SQL systems, and BI tools.

---

## Scenario-Based Question

### Scenario:

A manager asks for a monthly report that should be shared but not edited.

What format should an analyst provide?

**Suggested Answer:**

The analyst should export the Excel report as a PDF because it preserves formatting and prevents accidental editing.
